## Проблема

Ты заходишь в PUG-рейд. Танк объясняет тактику — по-испански. Хилер спрашивает что-то — по-немецки. Ты говоришь по-русски (или по-английски). Никто никого не понимает. Начинается пулл, люди умирают, и кто-то пишет "gg noob" — единственную фразу, которую знают все.

**Это происходит постоянно** в кроссерверных и кроссрегиональных группах WoW. Языковой барьер ломает координацию, вызывает вайпы и портит удовольствие от игры.

## Решение

BabelChat переводит чат WoW **в реальном времени**. Мини-аддон перехватывает сообщения из игры; приложение-компаньон переводит их через DeepL и показывает результат в стильном оверлее поверх WoW.

**Оригинальное сообщение ты видишь мгновенно. Перевод появляется через 0.5–2 секунды.**

Частые фразы ("gg", "ty", "ready?", "pull") переводятся мгновенно из встроенного разговорника — без обращения к API, без задержки. Полные предложения идут через DeepL и приходят за 1–2 секунды. Одно и то же сообщение никогда не переводится дважды (кэш).

### Когда BabelChat полезен?

- **Кроссерверные PUG-группы** — понимай тактику испанского танка, вопросы немецкого хилера
- **Международные гильдии** — следи за гильдчатом на своём языке без "english pls"
- **Игра на иностранных серверах** — зашёл на французский или корейский реалм? Чат теперь понятен
- **Рейд-лидерство** — давай команды на своём языке, игроки видят их на своём (через исходящий переводчик)
- **Шёпоты от незнакомцев** — пойми то случайное сообщение на португальском

## Возможности

- **Стриминг перевода** — оригинал появляется мгновенно, перевод подгружается через 0.5–2с
- **Автоопределение языка** — офлайн, ~1мс на сообщение (lingua-py)
- **22 языка перевода** — EN, RU, DE, FR, ES, IT, PT, PL, NL, SV, DA, FI, CS, RO, HU, BG, EL, TR, UK, JA, KO, ZH
- **Умный оверлей** — тёмная тема WoW, цвета каналов, прозрачный для кликов, перемещаемый
- **Двусторонний перевод** — переводи входящий чат И составляй исходящие сообщения на любом языке
- **Встроенный разговорник** — 45 фраз + 30 игровых аббревиатур без API
- **Словарь WoW** — 314 игровых терминов (lfm, wts, dps, tank и т.д.) на 14 языках
- **Фильтры каналов** — Группа, Рейд, Гильдия, Сказать/Крик, Шёпот, Подземелье
- **DeepL Free API** — 500 000 символов/месяц бесплатно (~10К сообщений)
- **Кэш переводов** — потокобезопасный SQLite + LRU, одно и то же не переводится дважды
- **Горячие клавиши** — переключай перевод, не выходя из игры
- **Установка аддона в один клик** — мастер настройки всё сделает

## Почему перевод занимает 0.5–2 секунды?

BabelChat использует **progressive rendering** (стриминг):

1. **Оригинальное сообщение ты видишь сразу** (0мс задержки)
2. **Перевод появляется под ним** когда DeepL ответит (0.5–2с)

Задержка — это round-trip до серверов DeepL: текст отправляется, переводится нейросетью, возвращается. Это та же задержка, что у Google Translate или любого облачного переводчика.

**Мгновенно (без задержки):**
- Игровые аббревиатуры: `gg`, `ty`, `brb`, `afk`, `wp`, `lol` — из разговорника
- Частые фразы: "hello", "thanks", "ready?", "good game" — из разговорника
- Повторные сообщения — из кэша
- Сообщения на твоём языке — показываются без перевода

**0.5–2с:**
- Полные предложения на иностранных языках — нужен вызов DeepL API
- Первое вхождение любой фразы — потом кэшируется

## Как это работает

```
┌──────────────────────────────────────────────────────────┐
│  World of Warcraft                                       │
│                                                          │
│  Аддон BabelChat                                         │
│  ├── Перехватывает события CHAT_MSG_* через WoW API      │
│  ├── Кольцевой буфер (50 сообщений)                     │
│  └── Пишет в BabelChatDB.wctbuf (Lua SavedVariable)     │
└──────────┬───────────────────────────────────────────────┘
           │  ReadProcessMemory (каждые 250мс)
           ▼
┌──────────────────────────────────────────────────────────┐
│  Приложение-компаньон (Python, от админа)                │
│                                                          │
│  Memory Reader ──→ Парсер ──→ Детектор языка             │
│       │                           │                      │
│       │    Разговорник (мгновенно)─┤                     │
│       │    Кэш (мгновенно) ───────┤                      │
│       │    DeepL API (0.5-2с) ────┤                      │
│       │                           ▼                      │
│       └───────────────→ Оверлей (PyQt6)                  │
└──────────────────────────────────────────────────────────┘
```

### Зачем нужно приложение-компаньон?

Lua-песочница WoW **не может делать HTTP-запросы**. Аддон перехватывает чат и показывает UI, но не может вызвать DeepL. Приложение-компаньон решает эту проблему, читая буфер аддона из памяти процесса.

Тот же подход используют **WeakAuras Companion** и **WarcraftLogs** — чтение памяти, полное соответствие ToS Blizzard.

## Установка

### Быстрый старт

1. Скачай `BabelChat.zip` из [Releases](https://github.com/Yumash/BabelChat/releases)
2. Распакуй и запусти `BabelChat.exe` **от имени Администратора**
3. Следуй мастеру настройки (получи [бесплатный ключ DeepL](https://www.deepl.com/pro-api), укажи путь к WoW, установи аддон)
4. Запусти WoW, зайди в группу — переводы появятся автоматически

### Из исходников

```bash
git clone https://github.com/Yumash/BabelChat.git
cd BabelChat
pip install -r requirements.txt
python -m app.main  # запуск от Администратора
```

### Аддон WoW (ручная установка)

Скопируй `addon/BabelChat/` в `World of Warcraft/_retail_/Interface/AddOns/BabelChat/`

### Как добавить новый термин

Отредактируй файл `addon/BabelChat/Data/*.lua`:

```lua
["newterm"] = {
    enUS = "English translation",
    esES = "Traducción española",
    ruRU = "Русский перевод",
    deDE = "Deutsche Übersetzung",
    frFR = "Traduction française",
    -- ... (14 языков)
},
```

## Соответствие ToS Blizzard

| Аспект | Статус |
|--------|--------|
| Чтение памяти | Только чтение. Как WeakAuras Companion, WarcraftLogs |
| Оверлей поверх игры | Разрешено. Как Discord Overlay |
| Addon API | Стандартные хуки CHAT_MSG_*. Используются всеми чат-аддонами |
| Нет инъекций | Нет DLL injection, нет hooking, нет записи в память WoW |
| Нет автоматизации | Исходящий перевод через буфер обмена (ручная вставка Ctrl+V) |

## Ограничения

- **Только Windows** — ReadProcessMemory это Windows API
- **Нужны права Администратора** — для чтения памяти
- **Лимит DeepL Free** — 500К символов/месяц (~10К сообщений). Есть платные планы
- **Исходящие сообщения** — копируй → вставляй в чат WoW (осознанно, для соответствия ToS)
